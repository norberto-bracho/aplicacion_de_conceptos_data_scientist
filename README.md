# aplicaci-n_de_conceptos_data_scientist
la finalidad de este repositorio es la práctica de mis conceptos adquiridos en mi formación constante en la ciencia de datos. como limpieza de datos para la correcta gestión de datos; conceptos estadísticos de medidas centrales y de dispersión como el desvío estandar, varianza, media, mediana, moda, coeficiente de variación. y la implemetación de graficos para representar visualmente los datos y encontrar posibles sesgos y correlaciones de datos.
